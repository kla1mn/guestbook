## Что реализовано
### Техническая реализация
- [x] Статический фронт, доступный из интернета. Реализован с помощью Object Storage. В UI приложения видно версию.
- [x] Есть реплики бэкенда, реализованные либо с помощью Serverless Functions. В UI приложения видно версию.
- [ ] Каждый раз попадаем на разные реплики бэкенда за счет скрытия бэкенда за API Gateway.
- [x] Данные приложения сохраняются в Serverless YDB.
- [x] Есть публичный доступ из интернета по HTTPS через бесплатные служебные домены Yandex Cloud.
